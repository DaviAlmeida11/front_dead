# Deadpool Front-End Project

Este é um projeto de estudo focado em **HTML**, **CSS** e **JavaScript**, inspirado no universo do **Deadpool**. O objetivo é praticar conceitos de front-end e criar uma página interativa e estilizada.

##  Tecnologias Utilizadas

- **HTML5** – Estrutura da página
- **CSS3** – Estilização e layout
- **JavaScript (JS)** – Interatividade e manipulação de elementos
- **Git & GitHub** – Controle de versão

##  Objetivos do Projeto

- Praticar a criação de layouts responsivos
- Trabalhar com cores, fontes e imagens de forma criativa
- Implementar pequenas interações com JavaScript
- Organizar o código de forma limpa e semântica
